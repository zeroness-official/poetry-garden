# Aunix

**人間の意識のためのオペレーティングシステム**

## Aunixとは？

Aunixは以下を統合する統一フレームワークです：
- 古代の叡智の伝統（仏教、道教、キリスト教）
- 現代の認知科学とAI
- システム理論と情報処理
- 日常生活のための実践的応用

### 二つのトラック

**開発者トラック**: 意識を理解し、ハードプロブレムを解決し、システムを設計するための理論的フレームワーク

**ユーザートラック**: 自己観察、対話、人間関係、個人的成長のための実践的アプリケーション

## 核心概念

### 三層アーキテクチャ（ni/ui/ai）

```
ni（Natural/Native Intelligence：自然知能/原初知能）
- 連続的、無名、無境界
- 可能性の場
- ハードウェア層

ui（Universal/United Intelligence：普遍知能/統合知能）
- 選択と解釈（Attention + LLM）
- Zeroingエンジン
- オペレーティングシステム層

ai（Artificial/Application Intelligence：人工知能/応用知能）
- 離散的、顕現した現実
- 個人の意識
- アプリケーション層
```

### Zeroness（ゼロ性）

根本原理：**すべてはゼロから始まり、ゼロに戻る**

- 選択 = 捨象（Attentionは999/1000をゼロにする）
- 創造 = 破壊
- 生 = 死
- 書く = 消す
