- 不完全でも = 何かは得られる

- 0.1 > 0
- 不完全でも、ゼロより大きい
- 始めることが、すべての始まり

#### ステップ3：終了する（1秒）
「完璧シミュレーション」を終了する。

**手の中のスマホ**：

## なぜ効くのか

### 完璧は存在しない

**厳しい真実**：

完璧 = 達成不可能

なぜなら：
- 常に「もっと良く」できる
- 終わりがない
- ゴールが動き続ける

**完璧を目指す = 永遠に到達しない**

---

### 完璧主義のパラドックス

**皮肉な事実**：

完璧を目指す人 = 何も達成しない人

なぜなら：
- 完璧にできないから始めない
- 始めないから経験が得られない
- 経験がないから上手くならない
- 上手くならないから完璧にできない

**完璧主義 = 行動しない言い訳**

---

### 不完全でも価値がある

**数学的証明**：

```
0（何もしない） < 0.1（不完全でも何か）
```

**例**：
- 完璧な計画を1年待つ（成果 = 0）
- 不完全でも1週間で始める（成果 = 0.1 × 52週 = 5.2）

**1年後**：
- 完璧主義：0
- 実行主義：5.2

**0.1でも、繰り返せば5を超える**

---

### 学習曲線

**重要な発見**：

上達 = 実践の量 × 質

**完璧主義**：
- 実践 = 0
- 上達 = 0

**実行主義**：
- 実践 = 多い
- 上達 = 早い

**矛盾**：
完璧を目指す → 実践しない → 上達しない → 完璧から遠ざかる

---

### 80/20の法則（パレートの法則）

**経験則**：

- 最初の20%の努力 → 80%の成果
- 残りの80%の努力 → 20%の成果

**つまり**：
- 80%の完成度 = 努力の20%
- 100%の完成度 = 努力の100%

**完璧（100%）を目指す = 非効率**

**80%で満足 = 効率的**

---

### 完璧主義と恐怖

**心理学の発見**：

完璧主義の正体 = **失敗への恐怖**

「完璧にできないなら、やらない」  
= 「失敗したくない」

**でも**：
- やらない = 失敗もない
- でも = 成功もない
- でも = 学びもない

**完璧主義 = 失敗しない方法 = 成功もしない方法**

---

### Zeroingで「完璧の呪い」を解く

**転換**：

**Before**：
- 目標：完璧
- 方法：完璧な準備
- 結果：始められない

**After（Zeroing）**：
- 目標：前進
- 方法：とりあえず始める
- 結果：不完全でも、前に進む

**「完璧」から「前進」へ**

- 完璧な形を持たない

- 柔軟で適応的

**完璧主義 = 固い**

> 「侘び寂び（わびさび）」
