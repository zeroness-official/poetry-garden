## Zeronessの3つの技術

### 技術1：Zeroing - 手放す3秒

**基本形**：

```
1秒：気づく
「今、何が動いている？」

1秒：認識する
「これは不要？必要？」

1秒：終了する
「不要なものは、終了。」
```

**使うとき**：
- 頭の中が混乱している
- 不安や心配が止まらない
- 複数のことが気になる
- 過去や未来に囚われている

**効果**：
- 頭がすっきりする
- 心が軽くなる
- 今に集中できる

---

### 技術2：Oneing - 戻る3秒

**基本形**：

```
1秒：今の自分を見る
「今、どう感じている？」

1秒：ありのままを受け入れる
「これが、今の自分」

1秒：自分に戻る
「これで、いい」
```

**使うとき**：
- 他人と比較している
- 自分を否定している
- 完璧を求めている
- 自信がない

**効果**：
- 自分を受け入れられる
- 比較から解放される
- ありのままでいられる

---

### 技術3：Zoning - 循環する3秒

**基本形**：

```
1秒：手放す（Zeroing）
「雑念を、手放す」

1秒：戻る（Oneing）
「今、これに戻る」

1秒：繰り返す（Zoning）
「また手放す、また戻る」
```

**使うとき**：
- 集中したい
- フロー状態に入りたい
- 長時間作業する
- クリエイティブな仕事

**効果**：
- 深い集中状態に入れる
- 時間を忘れて没頭できる
- 高いパフォーマンス

## 基本の3秒：場面別

### 朝、目覚めたとき

**Zeroing（手放す）**：
```
気づく：「今日の不安が、もう始まっている」
認識する：「まだ起きてもいないことのシミュレーション」
終了する：「今日は今日。不安を終了。」
```

**Oneing（戻る）**：
```
今の自分：「今、どう感じている？」
受け入れる：「これが、今の自分の状態」
戻る：「イマ・ココ・オノレに戻る」
```

**Zoning（循環）**：
```
呼吸：吐く（Zeroing）→ 吸う（Oneing）→ 繰り返す
リズム：今日の始まりのリズムを作る
循環：この循環で、一日を始める
```

---

### 仕事中

**Zeroing（手放す）**：
```
気づく：「気が散っている。3つのことを考えている」
認識する：「複数のシミュレーションが同時起動」
終了する：「今は、これだけ。他は終了。」
```

**Oneing（戻る）**：
```
一つを選ぶ：「今、本当に大事なのは？」
一つに戻る：「これだけに100%」
一つに集中：「ありのままの自分で、これをやる」
```

**Zoning（循環）**：
```
手放す：雑念が来る → Zeroing
戻る：作業に戻る → Oneing
繰り返す：この循環でフローへ
```

---

### 夜、寝る前

**Zeroing（手放す）**：
```
気づく：「今日の失敗が、頭を回っている」
認識する：「過去のリプレイ。変えられない」
終了する：「今日は終わった。明日は明日。終了。」
```

**Oneing（戻る）**：
```
今の自分：「今日、自分はよくやった」
受け入れる：「完璧じゃなくても、これが自分」
戻る：「今、ここ、この自分でいい」
```

**Zoning（循環）**：
```
一日の終わり：Zeroing（今日を手放す）→ Oneing（今の自分）
睡眠への準備：リズムを整える
明日へ：また新しい循環が始まる
```

## よくある質問

### Q1：3つの技術、どれから始めればいい？

**A：Zeroingから**

順番：
1. **Zeroing**（手放す）- まず不要なものを捨てる
2. **Oneing**（戻る）- 次に自分に戻る
3. **Zoning**（循環）- 最後に循環を作る

理由：
- Zeroingができないと、Oneingも難しい
- OneingができないとZoningも難しい
- 順番に学ぶのが自然

---

### Q2：3つ全部やらないとダメ？

**A：いいえ**

**状況によって使い分け**：
- 混乱している → Zeroing
- 自分を見失っている → Oneing
- 集中したい → Zoning
- 深く入りたい → 3つ全部（循環）

**基本**：
- 最初はZeroingだけでOK
- 慣れたらOneingを追加
- さらにZoningを追加

---

### Q3：うまくいきません

**A：それでOK**

Zeronessは練習。最初はうまくいかない。

**でも**：
- 気づいただけでOK
- 試しただけでOK
- 続ければ上達する

**完璧を求めない = Zeroing**

---

### Q4：どれが一番大事？

**A：全部大事、でも順番がある**

**3つは三位一体**：
- Zero（手放す）なしに、One（自分）に戻れない
- One（自分）なしに、Zone（循環）は作れない
- Zone（循環）なしに、深く生きられない

**すべてがつながっている**：
- Zeroing → Oneing → Zoning → Zeroing...
- これが循環
- これがZeroness

---

### Q5：Zoningは難しそう

**A：最初は難しい、でも自然になる**

**段階**：
1. **意識的な循環**（最初）
   - 手放す（意識的）→ 戻る（意識的）→ 繰り返す（意識的）

2. **半自動的な循環**（慣れてきたら）
   - 手放す → 戻る → 自然に繰り返す

3. **自動的な循環**（上級）
   - 呼吸のように
   - 意識しなくても
   - ずっと循環している

**Zoningは、呼吸のようなもの**
