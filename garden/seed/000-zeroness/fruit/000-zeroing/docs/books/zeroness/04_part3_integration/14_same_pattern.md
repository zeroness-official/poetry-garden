## 共通パターンの発見

### パターン：アプリの終了し忘れ

**すべての問題の正体**：

| 問題 | 起動しているアプリ |
|------|------------------|
| 通知に振り回される | 「通知待機アプリ」 |
| SNSがやめられない | 「無限スクロールアプリ」 |
| 返信が来なくて不安 | 「不安シミュレーションアプリ」 |
| 過去が頭から離れない | 「過去再生アプリ」 |
| 未来が不安 | 「未来予測アプリ」 |
| 集中できない | 複数アプリ同時起動 |
| 完璧主義で動けない | 「完璧シミュレーションアプリ」 |
| 比較してしまう | 「比較アプリ」 |
| 怒りが収まらない | 「怒りリプレイアプリ」 |
| 自信が持てない | 「過去失敗データベースアプリ」 |

**すべて、アプリを終了していないだけ。**

- 集中 → 複数のことをシミュレート
- 完璧 → 完璧なシナリオをシミュレート
- 比較 → 他人との差をシミュレート
- 怒り → 正義のシナリオをシミュレート
- 自信 → 失敗のシナリオをシミュレート

**すべて、シミュレーション。**  
**すべて、終了し忘れ。**
