# 付録A：Zeroingチェックリスト

---

## 日常のZeroingチェックリスト

### 朝のルーティン

```
□ 目覚めたとき
  - 今日の不安シミュレーションに気づく
  - 3秒Zeroing
  - 今日は今日

□ 朝食の前
  - スマホを見る前に3秒呼吸
  - 本当に今、必要？
  - 必要なければ後で

□ 出かける前
  - 今日のタスクリスト確認
  - 本当に全部必要？
  - 不要なものをZeroing
```

---

### 仕事中のルーティン

```
□ 仕事開始時
  - 複数のことを考えていないか
  - 今、これだけに集中
  - 他は全部Zeroing

□ 通知が来たとき
  - 3秒待つ
  - 今、必要？
  - 必要なければ後で

□ 集中が切れたとき
  - 何に気を取られている？
  - それは今必要？
  - Zeroingして戻る

□ 昼休み前
  - 午前の仕事をZeroing
  - 脳をリセット
  - 新鮮な午後へ
```

---

### 夜のルーティン

```
□ 帰宅時
  - 仕事モードをZeroing
  - 家モードへ切り替え
  - 境界を明確に

□ 夕食後
  - SNSチェック時間設定
  - 10分だけ
  - タイマー終了でZeroing

□ 寝る前
  - 今日1日をZeroing
  - 良いことも悪いことも
  - 明日は明日

□ ベッドで
  - 今日の反芻が始まったら
  - 3秒Zeroing
  - 身体感覚に注目
```

## 緊急時チェックリスト

### パニック・強い不安

```
□ 気づく
  「今、パニックになっている」

□ 身体に注目
  □ 呼吸を感じる
  □ 足の裏を感じる
  □ 今ここに戻る

□ 3秒Zeroing × 3回
  □ 1回目：気づく
  □ 2回目：認識する
  □ 3回目：終了する

□ ゆっくり呼吸
  □ 4秒吸う
  □ 4秒止める
  □ 4秒吐く
  □ 3回繰り返す
```
