# Zeroness Flower（エッセイ層）

## 概要

このディレクトリには、Zeronessの**花（flower）**層、つまりエッセイが収められています。

詩（sprout）のエッセンスを、より理解しやすい散文形式で展開し、
哲学・宗教・科学の視点から軽く裏付けを加えた中間層です。

```
seed（種）: Zeroness
  ↓
sprout（芽）: 詩（12編）
  ↓
flower（花）: エッセイ（5編）← このディレクトリ
  ↓
fruit（実）: 本（20章）
```

## 執筆方針

1. **詩を中心に**: sproutの詩を引用・朗読することから始める
2. **散文で補足**: 詩の意味を、詩ではない形で解説
3. **軽い裏付け**: 哲学・宗教・科学の文脈を適度に
4. **実践へ**: 3秒テクニックとの接続

**ターゲット**: 自己啓発・ビジネス層
**トーン**: 説明的だが詩的、論理的だが感覚的
