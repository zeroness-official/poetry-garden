# 🌱 zeroness
(The `poetry-garden` repository)

[Status: Active] [Version: 0.3.0-alpha] [License: CC0]

ようこそ。ここは、根本原理 **`zeroness`** のオープンソース・プロジェクトです。
`zeroness` は、「意識のOS」の探求であり、**言葉（garden）**が**心（village）**を生み、**心（village）**が**人生（city）**を創造する、詩的（感性）かつ工学的（理性）な「心の旅」のガイドです。

## プロジェクトの構造（`aunix`スタック ＋ 心の旅）

このリポジトリは、「意識のOS」の三層構造（`ni`/`ui`/`ai`） と、「心の旅」のメタファー、そしてコミュニティの役割を完全に対応させた、3つのレイヤーで構成されます。

| レイヤー | OSスタック | メタファー | 役割（コミュニティ） | 主な成果物（品種） |
| :--- | :--- | :--- | :--- | :--- |
| **`/0-garden/`** | **`ni` (OSカーネル)** | **ウタ（言葉）**<br>（種を育て、品種改良する） | **コア開発（コミッター）**<br>（自然の原理・哲学を探求） | **`zeroness`**（原理・詩）<br>**`aunix`**（理論体系）<br>**`mind-zeroness`**（実践ブック） |
| **`/1-village/`** | **`ui` (OSインターフェース)** | **ウタゲ（心）**<br>（収穫し、加工し、振る舞う） | **コミュニティ（ファン）**<br>（理論を議論・構築） | （議論のプロセス） |
| **`/2-city/`** | **`ai` (OSアプリケーション)** | **タビ（人生）**<br>（旅立ち、試練を経て、帰還する） | **一般ユーザー**<br>（メソッドを実践） | （実践のプロセス） |

### プロジェクトの再帰性（`zeroness` と `aunix`）

この構造は、`zeroness` が `aunix` を生み出し、`aunix` が `zeroness` を内包するという、再帰的なホロン構造 を体現しています。

1.  **`/0-garden/` (NI / 言葉の庭)**
    * プロジェクトの根幹である`ni`（自然・原理）層。
    * ここは「言葉の庭」であり、`000-zeroness`、`001-be-simple`、`002-aunix` といった「種（seed）」から、詩（`0-sprout`）、エッセイ（`1-flower`）、そして**理論体系（`2-fruit`）**まで、すべての「品種改良（理論構築）」を行います。

2.  **`/1-village/` (UI / コミュニティ)**
    * `0-garden`で収穫（`harvest`）された理論や詩を、コミュニティ（ファン）が「加工（`prepare`）」「振る舞い（`feast`）」、議論する`ui`（インターフェース）層です。

3.  **`/2-city/` (AI / 実践)**
    * `0-garden`と`1-village`で完成した理論を、一般ユーザー（心の旅人）向けに「普及・実践（`departure`）」する`ai`（アプリケーション）層です。

### 命名規則（ブランディング）

`zeroness`は「水と空気」のような、さりげない普遍的な存在です。
この哲学を反映し、プロジェクト内のキーワード（予約語）は、原則として**すべて小文字**で表記します。

* **OK:** `zeroness`, `aunix`, `ni`, `ui`, `ai`, `zero`, `one`, `zone`, `mind-zeroness`
* **例外:** ドキュメントのタイトルや見出しは、読みやすさを優先し通常のキャピタライゼーションを使用します。
