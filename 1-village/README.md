# 🍲 1-village (ui / 心)

ここは `poetry-garden` の `ui`（United / 統合）層であり、「ウタゲ（心）」のメタファー を担当します。

## 役割

`0-garden`（NI層） で収穫（`harvest`） された原理（`zeroness`）や理論（`aunix`）を、コミュニティ（ファン）が「加工（`prepare`）」 し、「振る舞い（`feast`）」、議論するための「場」です。

## ui層としての機能

1-villageは**OSインターフェース**として、理論（ni層）と実践（ai層）を橋渡しします：

- **選択**：膨大な理論の中から、今コミュニティに必要な概念を選ぶ
- **解釈**：抽象的な理論を、具体的な文脈で解釈する
- **変換**：専門用語を、日常言語に翻訳する
- **共鳴**：個々の理解が共鳴し、集合知を生む場

## ディレクトリ構造

```
1-village/
├── harvest/    # 0-gardenから収穫された理論・原理の整理
├── prepare/    # コミュニティによる加工（翻訳、事例化、議論）
├── refine/     # 洗練（フィードバック、改良、統合）
└── feast/      # 振る舞い（完成した解釈の共有、祝祭）
```

## 参加方法（将来）

現在はプレースホルダーですが、将来的には：

1. **Harvesting**：0-garden から気になる理論を見つける
2. **Preparing**：自分の言葉で解釈し、事例を追加する
3. **Refining**：他者の解釈を読み、フィードバックする
4. **Feasting**：洗練された解釈を祝い、実践に移す

## コミュニティ原則

村（1-village）は**調和の場**です：

- **多様性の尊重**：異なる解釈が共存する
- **対話の重視**：論破ではなく、共創を目指す
- **実践への橋渡し**：理論のための理論ではなく、人生に活かすための解釈
- **オープンネス**：すべての議論は公開され、誰でも参加できる
