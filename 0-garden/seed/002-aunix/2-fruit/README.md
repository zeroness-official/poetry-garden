ここは `002-aunix` の「言実（fruit）」です。
`aunix`は、`zeroness`カーネル（ni層）を工学的に説明する理論体系（OSドキュメント）であり、それ自体が再帰的な三層構造を持ちます。

* **/ni/ (kernel):** カーネル理論
    * `THEORY.ja.md`: OSの完全な理論仕様書。
* **/ui/ (api):** インターフェース
    * `API.md`: OSの5つの主要システムコール（API）定義。
* **/ai/ (apps):** 応用
    * `PRACTICE.ja.md`: APIの基本的な実践ガイド。
    * `embodied-ai.md`: 工学的応用例（身体性AI）。
