# 🌱 garden (ni / 言葉)

ここは `poetry-garden` の `ni`（Natural / OSカーネル）層であり、「ウタ（言葉）」のメタファー を担当します。

## 役割

`garden`は「農業試験場」です。プロジェクトの根幹である`ni`（自然・原理）層として、`zeroness` や `aunix` といった「種（seed）」から、詩（`sprout`）、エッセイ（`flower`）、そして**理論体系（`fruit`）**まで、すべての「品種改良（理論構築）」を行います。

## 成果物（品種）

* `/garden/seed/000-zeroness/`: 根本原理 `zeroness` と、その実践版 `zeroing-book`。
* `/garden/seed/001-be-simple/`: 普及版の哲学 `be-simple`。
* `/garden/seed/002-aunix/`: 工学理論 `aunix` と、その理論仕様書（`THEORY.ja.md`）およびAPI（`API.md`）。
