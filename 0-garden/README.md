# 🌱 0-garden (ni / 言葉)

ここは `poetry-garden` の `ni`（Natural / OSカーネル）層であり、「ウタ（言葉）」のメタファー を担当します。

## 役割

`0-garden`は、`zeroness`の根幹概念が豊かに育っていく「言葉の庭」です。

プロジェクトの根幹である`ni`（自然・原理）層として、`zeroness` や `aunix` といった「種（seed）」から、詩（`0-sprout`）、エッセイ（`1-flower`）、そして**理論体系（`2-fruit`）**までを育み、収穫します。ここは「農業試験場」でもあり、すべての「品種改良（理論構築）」を行う場でもあります。

## 成果物（品種）

* `/0-garden/seed/000-zeroness/`: 根本原理 `zeroness` と、その実践版 `zeroing-book`。
* `/0-garden/seed/001-be-simple/`: 普及版の哲学 `be-simple`。
* `/0-garden/seed/002-aunix/`: 工学理論 `aunix` と、その理論仕様書（`THEORY.ja.md`）およびAPI（`API.md`）。
